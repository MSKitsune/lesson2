# Подсказка по GIT

Создание репозитория:
```sh
git init
```

Создание коммитов add
```sh
git add
git add <имя файла>
```

Создание коммитов
```sh
git commit
```

Посмотреть лог в полном виде
```sh
got log
```

Посмотреть лог в сокращенном виде
```sh
git log --oneline
```

Перемещение по веткам
```sh
git checkout <имя ветки>
```

Отображение всех веток
```sh
git branch
```
